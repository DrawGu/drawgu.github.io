## 博客搭建过程记录

求快先Google了一下，直接在Github Pages建博客的方案：Hexo、Jekyll、Hugo等。我机器上有node环境，直接选Hexo。

### 安装

```shell
npm install hexo-cli -g
hexo init <folder>
cd <folder>
yarn
```

### 写博客
使用`hexo new`命令创建文章
